## Performance

**What is Performance?**  
**Key Terms:** Efficiency, Speed  

- **Core Question:**  
  "How fast can the system finish a task?"

- **Definition:**  
  Refers to how quickly and efficiently a system can finish tasks under the given current conditions.

- **Characteristics:**
  - Response time
  - Throughput
  - Latency
  - Resource utilization
  - Efficiency

- **Measured When:**
  - The system has manageable users and data volume
  - Current resource constraints might pose a problem
  - User experience optimization is prioritized
  - Response times need to meet SLA requirements

## Scalability

**What is Scalability?**  
**Key Terms:** Capacity, Growth  

- **Core Question:**  
  "Can the system grow to handle more work?"

- **Definition:**  
  Refers to a system's ability to handle increased workload while maintaining the same performance levels (or even improving).

- **Characteristics:**
  - Elasticity
  - Capacity planning
  - Resource scaling
  - Load distribution
  - Architectural flexibility

- **Types of Scaling:**
  - **Vertical Scaling**
  - **Horizontal Scaling**
  - **Diagonal Scaling**
